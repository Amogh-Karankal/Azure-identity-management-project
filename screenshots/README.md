## How to Take Good Screenshots

1. **Clean up the interface** - Close unnecessary tabs/panels
2. **Highlight relevant info** - Use browser zoom if needed
3. **Remove sensitive data** - Blur or crop passwords, tenant names if desired
4. **Use consistent naming** - Follow the naming convention above
5. **PNG format** - Better quality than JPG for UI screenshots

## Tools for Screenshots

- **Windows:** Snipping Tool, Win+Shift+S
- **Mac:** Cmd+Shift+4
- **Browser:** Built-in screenshot tools
- **Annotation:** Snagit, Greenshot, or built-in tools
